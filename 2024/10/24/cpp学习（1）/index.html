#引用与指针
##引用部分
什么是引用？

引用是相当于一个变量的别名,有两种引用类型，一个是常量引用，另一个则是非常量引用

为什么使用引用？

因为它是高效安全易用的

怎么使用引用？

###（1）引用的初始化和声明

{Type& name(var);//非常量引用
const Type& name(var);//常量引用
/*
在使用引用作为函数传参的方法时，非常量引用可以在函数内修改原始对象的值，即通过引用修改其引用的原始对象
除有以上需求外，最好使用常量引用进行传参
*/}
###（2）引用可以用来传递函数参数等等,例子如下
```cpp
{test(int& a){
++a;
//下面的main函数内定义的b的值也会改变
}
test1(const& a){
    ++a;//不被允许，因为使用的是常量引用
}
test2(){
    const int d{1};
    int& a{d};//无法引用绑定到d，因为d是不可修改的左值
    const int& c{d};//可以绑定到常量d
    c=2;//无法用常量引用修改对象
    int e{6};
    int& f {e};
    f=7;//e=7
    const int& g{e};
    g=8;//无法用常量引用修改对象
}
int main (){
    int b{5};
    test(b)
}
/*
基本数据类型可以直接按值传递，因为其传值的开销很小，
而类或结构体等类型，传值复制开销高，因此使用（常量）引用传递
*/}
### Tips:
左值引用只能绑定到可修改的左值。<br>
对常量的左值引用可以绑定到可修改的左值、不可修改的左值和右值。因此可以灵活使用<br>

##指针部分（原始指针部分知识）
```cpp
{#include <iostream>
int main() {
    int num{ 6 };
    int num1{ 1 };
    std::cout << &num <<"\n";//&是取地址操作，&num 获取变量num所存储的内存地址
    std::cout << *(&num) << "\n";//*是解引用运算符，可以用来访问内存地址所存储的值

    int* ptr{ &num };//这是一个指向变量num的指针变量，保存着num的地址
    std::cout << *ptr << "\n";//打印ptr指向的值
    std::cout << ptr << "\n";//打印的是指针变量ptr所指向对象的内存地址

    *ptr = 9;//修改ptr所指向的num的值
    ptr = &num1;//修改指针变量ptr所指向的对象
    std::cout << *ptr <<"\n";
    std::cout << ptr << "\n";

    std::cout << sizeof(ptr) << '\n'; // 指针变量的大小与指针所指向的类型无关，可执行文件的位数（32 or 64）决定指针变量大小

    //什么是悬空指针？例子如下
    int num3{ 6 };//当num3变量被销毁时，下面这个就成悬空指针了
    int* ptr1{ &num3 };
    //什么是空指针？
    int* ptr2{};
    int* ptr3{ nullptr };//nullptr是空指针的关键字，可以显式地初始化指针或者可以为指针变量附上null值
    //简单的判断指针是否为空指针，用if判断就行，if(xxx=nullptr)......

    //常量指针是指向常量的指针，指针常量说的是这个指针本身是一个常量
    int a{ 5 };
    int b{ 6 };
    int c{ 7 };
    const int* ptr4{&a};//这是常量指针（指向常量值的指针）
    *ptr4 = 55;//不能通过*来修改指针所指的常量
    ptr4 = &b;//不耽误修改指针所指向的对象

    int* const ptr5{&b};//这是指针常量（const指针）
    *ptr5 = 66;//可以
    ptr5 = &c;//这个指针本身是常量，所以不能修改指针所指向的对象

    const int* const ptr6{&c};//这是指向常量的常量指针


    return 0;
}}
